## Design Consistency Rules

1. **Typography**: Each template maintains its font family across all subsections
2. **Color Scheme**: Subsections use template's primary/accent colors
3. **Spacing**: Consistent padding and margins matching template style
4. **Icons**: Material Symbols Outlined used consistently
5. **Hover Effects**: Match template's interaction patterns
6. **Borders**: Follow template's border radius conventions

## Implementation Notes

- All subsections are responsive (mobile-first approach)
- Grid layouts adapt from 1 column (mobile) to 3-4 columns (desktop)
- Icons are consistently sized within each template
- Text hierarchy maintained throughout
- Accessibility considerations (semantic HTML, proper contrast)
