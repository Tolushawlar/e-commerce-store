# Custom Ecommerce Platform

A comprehensive multi-tenant ecommerce platform for creating custom online stores for clients with dedicated dashboards and super admin management.

## Features

### Client Features
- Custom branded ecommerce stores
- Product management dashboard
- Order management
- Customer management
- Analytics and reporting
- Store customization

### Super Admin Features
- Multi-client management
- Platform analytics
- User management
- Store template management
- Billing and subscriptions

## Tech Stack
- Frontend: HTML, CSS (Tailwind), JavaScript
- Backend: PHP
- Database: MySQL
- Design: Inspired by CampMart marketplace design

## Project Structure
```
/client-dashboard/     - Client store management interface
/super-admin/         - Super admin management panel
/store-templates/     - Customizable store templates
/api/                - Backend API endpoints
/assets/             - Shared assets (CSS, JS, images)
/database/           - Database schema and migrations
```

## Getting Started
1. Set up MAMP/XAMPP
2. Import database schema
3. Configure database connection
4. Access super admin panel at `/super-admin/`
5. Create client accounts and stores
